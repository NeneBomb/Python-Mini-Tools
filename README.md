# Python Mini-Tools Collection

Una colección de scripts pequeños y utilidades en Python que demuestran diferentes conceptos de programación, desde consumo de APIs hasta lógica de juegos simple.

---

## Fetch-Insult
Un script que consulta la API de `evilinsult.com` para obtener un insulto aleatorio. Demuestra el uso de fallbacks entre `requests` y `urllib`.

## Revolver Roulette
Un juego de azar con lógica de turnos y arte ASCII. Ideal para practicar el manejo de bucles y la librería `random`.

## Weather API
Script que consume la API de Open-Meteo para obtener el clima actual. Ejemplo de manejo de peticiones `HTTP` y formateo de `JSON`.
